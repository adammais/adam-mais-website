Multiplayer Jamaican Dominoes

Description

  • Adam Mais

  • Play here!

Developed a multiplayer Jamaican dominoes game for the web using Node.js, Socket.IO, and Phaser. Users can create or join a game room with the game code. The dominoes are randomly shuffled and delt. They can be dragged and dropped, and automatically orient themselves. Moves are propogated to the other players in the game room

Technology

  • HTML
  • JavaScript
  • Node.js
  • Phaser
  • Socket.IO
